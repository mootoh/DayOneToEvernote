DayOne to Evernote

A handy Automator workflow that imports your DayOne starred note to Evernote.

Usage

  1. Download the app
  2. Star your DayOne notes
  3. Launch DayOne to Evernote app
  4. Specify the index of the note to import (0 origin)

Setup

% gem install sqlite3
